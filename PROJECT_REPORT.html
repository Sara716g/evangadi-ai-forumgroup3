🟢 Register User

POST /api/auth/register
  • Creates unverified user in database
  • Hashes password with bcrypt
  • Generates 6-digit OTP code
  • Sends verification email
  • Returns success (no JWT until verified)

🟢 Verify Email

POST /api/auth/verify-email
  • Validates 6-digit OTP code
  • Marks user as verified (is_verified = TRUE)
  • Sends welcome email
  • Returns JWT token

🟢 Login

POST /api/auth/login
  • Verifies email and password
  • Blocks unverified users with clear message
  • Returns JWT token for verified users

🟢 Forgot Password

POST /api/auth/forgot-password
  • Generates reset code and sends email
  • Rate limited: 5 requests per 15 minutes
  • try/catch prevents 500 on SMTP failure

🟢 Verify Reset Code

POST /api/auth/verify-reset-code
  • Validates reset code before password change
  • Checks code hasn't expired

🟢 Reset Password

POST /api/auth/reset-password
  • Updates password with code verification
  • Hashes new password with bcrypt

🛡️ Security Features

🔐 Authentication

  • JWT tokens with secure signing
  • bcrypt password hashing (salt rounds)
  • SHA-256 for verification codes
  • Email verification required before login

🚫 Rate Limiting

  • Registration: 5 requests / 15 minutes
  • Forgot Password: 5 requests / 15 minutes
  • Verification: 10 requests / 15 minutes

🔒 Protected Routes

  • JWT middleware on all authenticated endpoints
  • Ownership verification for notifications
  • Prevent self-answering questions

📧 Email Security

  • try/catch on all sendEmail calls
  • Graceful SMTP failure handling
  • No sensitive data in error responses
